# Skill: Tutorial Quality Review

Review the final draft as a public tutorial, not as an internal artifact.

Check for:
- clear context before instruction
- useful step ordering
- table of contents and navigation
- omission of internal scaffolding
- whether the piece reads like a tutorial instead of organized notes
- whether incidental setup details are distracting from the core lesson
- whether the first actionable section starts with the real workflow instead of
  remote-access or recording context
